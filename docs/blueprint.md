# **App Name**: StudentHub AI

## Core Features:

- Login & Authentication: Secure user authentication system.
- AI-Powered Chatbot: Intelligent chatbot to assist students with their queries and give appropriate answers to the best of its abilities, acting as a tool with its reasoning enhanced by the student data at hand.
- Leave Management: System for students to apply for and manage leaves.
- Resume Builder: Tool to create professional resumes from templates.
- Admin Dashboard: Admin interface for managing users and content.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a trustworthy and professional feel.
- Background color: Light blue (#E3F2FD), a desaturated version of the primary color, to create a calm and clean backdrop.
- Accent color: Purple (#7E57C2) for highlighting important UI elements, such as buttons and links.
- Body and headline font: 'Inter', a grotesque-style sans-serif font for clear and modern readability.
- Use consistent, line-based icons to maintain a clean and modern look.
- Employ a clean and organized layout with a focus on intuitive navigation.
- Subtle transition animations and loading indicators to improve UX.
